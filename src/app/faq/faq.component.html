On the homepage, navigate to the sign in page by clicking the sign in button at the top right of the nav bar

View a list of all active users

1. On the homepage, click the person icon at the top right of the nav bar to reveal a dropdown menu and click on "User Configuration"

Create a New User

1. On the User Configuration Page, click the "create user" button below the table of users.

2. Fill out all of the fields for the new user's employee credentials. Once completed click the continue button.

2. Fill out all of the fields for the new user's personal information. Once completed click the "Create New User" button.

Update a User

1. On the User Configuration Page, click on the pencil icon for the user you want to update.

2. Review the current information for the selected user, and make updates any of the available fields.

2. After inputting changes, click the "Update User" button

Delete a User

1. On the User Configuration Page, click on the trash can icon for the user you want to delete.

2. On the pop-up box, confirm the user you want delete and click the "okay" button.
