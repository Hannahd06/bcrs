1. On the homepage, click the employee's button at the top right of the navbar to reveal a dropdown menu and Select "Sign in."

2. At the bottom left of the sign-in form, click on the "Register here" link.

3. Fill out the fields for the personal information. Once all items are filled, click the "Next" button.

4. Fill out the fields for the employee credentials. Once all items are filled, click the "Next" button.

5. Select from the available security questions in the dropdown menu. Make sure you select three security questions.

6. Provide an answer for each of the selected security questions.

7. Review security questions and answers and click the "submit" button. You have now registered.

View a list of all active users

1. On the homepage, click the person icon at the top right of the nav bar to reveal a dropdown menu, and click on "User Configuration."

Create a New User

1. On the User Configuration Page, click the "Create User" button below the table of users.

2. Fill out all of the fields for the new user's employee credentials. Once completed, click the "Continue" button.

3. Fill out all of the fields for the new user's personal information. Once completed, click the "Create New User" button.

Update a User

1. On the User Configuration Page, click the pencil icon for the user you want to update.

2. Review the selected user's current information and update any available fields.

3. After inputting changes, click the "Update User" button.

Delete a User

1. On the User Configuration Page, click the trash can icon for the user you want to delete.

2. On the pop-up box, confirm the user you want to delete and click the "Okay" button.
